AgriExpert — AI-Powered Agriculture Support 🌾

AgriExpert is an integrated ecosystem designed to bridge the gap between advanced technology and rural farming. By combining Agentic AI, real-time IoT monitoring, and a highly accessible Telugu-first interface, we empower farmers to make data-driven decisions in their native language.

🚀 Key Features
Agentic AI Crop Analysis: Instant diagnosis of crop diseases via image uploads or voice descriptions.

Regional Accessibility: Complete UI localization in Telugu and English.

Illiterate-Friendly Design: Integrated Telugu Voice Tour and audio instructions to assist farmers who cannot read or write.

IoT Soil Monitoring: Real-time moisture and health data pulled directly from Arduino sensors through a custom Python gateway.

Agricultural Marketplace: A secure platform to buy quality inputs and rent heavy machinery like tractors.

🛠️ Tech Stack
Frontend: React, Vite, TypeScript, Tailwind CSS, and shadcn/ui.

Backend: FastAPI (Python) for AI processing and service logic.

Database: SQL for structured data management.

Hardware: Arduino (Soil Moisture Sensors) & Python Serial communication.
